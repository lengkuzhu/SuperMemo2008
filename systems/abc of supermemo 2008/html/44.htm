Subject: Topic #44: To import a web article to SuperMemo: copy the selected text in your web browser (e.g. ...
To import a web article to SuperMemo:

  1. copy the selected text in your web browser (e.g. Ctrl+C)
  2. press Ctrl+Alt+N (in SuperMemo)

Ctrl+Alt+N is the same as Edit : Add a new article on the menu (if you select File : Level : Middle)

Try to import an article now!

For example: press Ctrl+Shift+W, type SuperMemo and press Enter to import an article about SuperMemo from Wikipedia.

You can also import articles from other sources. For example:

  1. Open any article in your web browser (for example, right click over this link www.supermemo.com/help/read.htm and choose Open in new window)
  2. Select any portion of text with the mouse (in the browser)
  3. Press Ctrl+C (in the browser)
  4. Return to SuperMemo (e.g. with Alt+Tab)
  5. Press Ctrl+Alt+N (in SuperMemo)
