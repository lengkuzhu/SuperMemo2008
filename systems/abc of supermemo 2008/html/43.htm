Incremental reading: Importing articles

The first step in incremental reading is to import an article to read and learn. Usually you will import an article directly from the Internet.

There are two basic ways of importing an article:

  1. One article - copy the selected text in your web browser to the clipboard, and press Ctrl+Alt+N in SuperMemo
  2. Many articles - open many articles in Internet Explorer, and press Shift+F8 in SuperMemo. You will be able to import selected articles

You can also import articles from Wikipedia with Ctrl+Shift+W

Topic #44: To import a web article to SuperMemo: copy the selected text in your web browser (e.g. ... (7 elements)

To import a web article to SuperMemo:

  1. copy the selected text in your web browser (e.g. Ctrl+C)
  2. press Ctrl+Alt+N (in SuperMemo)

Ctrl+Alt+N is the same as Edit : Add a new article on the menu (if you select File : Level : Middle)
